
GLXY UNCENSORED is a Discord bot that provides an "evil" AI assistant experience using a local language model.

## Features

- Responds to user queries with an "uncensored" AI personality
- Uses a local language model (LMStudio) for generating responses
- Includes startup animations and evil-themed message formatting
- Handles long responses by splitting them into multiple messages

## Setup

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Set up environment variables:
   - Rename `env.example` to `.env`
   - Update the `.env` file with your API keys and other configuration

### Getting API Keys and Discord Setup

1. Create a Discord Application:
   - Go to the [Discord Developer Portal](https://discord.com/developers/applications)
   - Click "New Application" and give it a name
   - Go to the "Bot" tab and click "Add Bot"

2. Get your Discord Bot Token:
   - In the "Bot" tab, find the "Token" section
   - Click "Copy" to copy your bot token
   - In your `.env` file, set `DISCORD_UNCENSORED_TOKEN=your_bot_token_here`

3. Create a Discord Server and Channel:
   - Create a new server in Discord (or use an existing one)
   - Create a new text channel for the bot

4. Get Channel ID:
   - Enable Developer Mode in Discord (User Settings > Appearance > Advanced > Developer Mode)
   - Right-click on the channel you created and select "Copy ID"
   - In your `.env` file, set `STARTUP_CHANNEL_ID=your_channel_id_here`

5. Invite the Bot to Your Server:
   - In the Discord Developer Portal, go to the "OAuth2" tab
   - In the "Scopes" section, select "bot"
   - In the "Bot Permissions" section, select the necessary permissions
   - Copy the generated URL and open it in a browser to invite the bot to your server

## Usage

- Start the bot: `python glxy_uncensored.py`
- Use the `/uncensored` command in Discord to interact with the AI
- If commands are not working, an admin can use `/sync` to update them
